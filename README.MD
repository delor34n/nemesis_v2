Proyecto "Némesis"
Sistema solar con cinturón de asteroides con enanas rojas.

Instrucciones como Ejecutar:

-> hacer doble clic a "Nemesis.sh"
-> esperar q se ejecute el programa

Botones:

F2 : 		  activa el sistema
F1 : 		  detener sistema
flecha arriba:    avanza en el eje Z
flecha abajo:     retrocede en el eje Z
flecha derecha:   avanza en el eje x
flecha izquierda: retrocede en el eje x
F7 :		  Némesis expande la onda y mueve los meteoros de la nube de Oort
